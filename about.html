Dash - Making JavaScript games fast, fun, and free!

We all love making games, but lets face it, it's not the most readable, or easy to write

That's why we created Dash!

An opensource, canvas based, JavaScript library for 2D Games

Set up your Dash project in as little as five minutes:

  • 1. Download the newest version of Dash
    • <script src="https://dash.netlify.app/dash.js"></script>
    Download
    By using Dash you agree to our license and terms

  • 2. Set up the game screen and create sprites
    • const canvas = new Canvas("canvas", 600, 400); const player = new Sprite(0, 0, 50, 50, "image", "blue", "image.png");
  • 3. Add all the sprites to the screen and start the program!
    • canvas.addSprite(player); canvas.start();
